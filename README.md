# 小程序疫苗预约网站系统SSM

## 前言

欢迎来到我们的小程序疫苗预约网站系统SSM项目。本项目旨在为广大用户提供一个便捷、高效的在线疫苗预约平台，通过微信小程序实现快速预约，后端采用SSM框架进行开发，确保系统的高效稳定。

## 内容介绍

本项目主要包括以下功能：

1. 用户注册、登录、个人信息管理
2. 疫苗信息展示、预约、预约时间查询
3. 预约成功通知、预约记录查询
4. 后台管理系统，包括用户管理、疫苗管理、预约管理等功能

为了提高用户体验，本项目采用了Vue、Uniapp等前端技术，实现了界面简洁、响应速度快的效果。

## 技术介绍

- 语言：Java
- 使用框架：Spring、Springmvc、MyBatis、微信小程序
- 前端技术：JS、Vue、CSS3、Uniapp
- 开发工具：IDEA/Eclipse、Uniapp
- 数据库：MySQL 5.7/8.0
- 数据库管理工具：phpstudy/Navicat
- JDK版本：jdk1.8
- Maven：apache-maven 3.8.1-bin
- 前端环境：Node.Js 12\14\16
